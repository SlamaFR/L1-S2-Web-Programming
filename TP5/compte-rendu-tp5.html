Exercice 1 : Hello World

  1. Comment fait-on pour ouvrir la console développeur ?

    On appuie sur la touche F12 ou la combinaison Ctrl + Alt + I.

  2. Afficher les chiffres de 0 à 9 dans la console.

    On utilise le code suivant :

    for (let $i = 0; $i < 10; $i++) {
    console.log($i)
}

  3. Modifier le code pour ne pas afficher les multiples de 3.

    On ajoute la condition suivante :

    if ($i % 3 === 0) continue;

  4. Écrire une fonction affichant les n premiers entiers avec n en paramètre.

    Code :

    function afficher_n_premiers_entiers($n) {
    for (let $i = 0; $i < $n; $i++) {
        console.log($i);
    }
}

    Cette ligne indique au navigateur qu'il doit utilise JavaScript en "strict mode".
    Ceci empêche d'utiliser des variables indéfinies et par conséquent force à rédiger du beau code avec du sens.
